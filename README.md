# sevencowcloud-shortvideo-service
2023年七牛云1024校园编程马拉松作品后端 🥺🥺

使用 `golang grpc` + `http-gateway` 开发。

## 运行方法

```bash
go mod tidy

go run cmd/server/main.go
```

or

```bash
make dev-up
make migrate-up
```
